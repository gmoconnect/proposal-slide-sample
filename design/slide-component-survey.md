## 提案書ノウハウ集への反映内容

以下の新知見を design/proposal-knowhow.md に追加済み:

1. **スライド作成の5ステップ** (§3.1)
   - 言語化→構造化→空パッケージ→ボディ→装飾の段階的プロセス
   - 手書き紙芝居による事前構想

2. **スライドの6構成要素** (§3.2)
   - タイトル、キーメッセージ、ボディ、脚注、出所、ページ番号

3. **キーメッセージの検証方法** (§3.4)
   - Why so? / So what? の双方向検証

4. **アジェンダスライドの戦略的活用** (§3.5)
   - 現在位置の視覚化、プロセス図によるナビゲーション

5. **Appendixの戦略的活用** (§3.6)
   - 突っ込まれそうなポイントの事前準備

6. **ボディの5大レイアウトパターン** (§3.9)
   - 並列/順序/分割/複合/4分割

7. **図表・チャートの体系的分類** (§3.10)
   - グラフ5種+チャート7種の使い分け
   - 円グラフを避けるルール
   - 差分矢印ラベル、Callout等の補助テクニック

8. **フォントサイズ下限ルール** (§5.2)
   - 16pt以上推奨、12pt以下は避ける

## Slidevテンプレート化に向けたコンポーネント設計

### スライドタイプ一覧（Slidevで実装すべきレイアウト）

| # | スライドタイプ | 用途 | 構成要素 |
|---|--------------|------|---------|
| 1 | **表紙（Cover）** | 提案書の表紙 | タイトル、サブタイトル、会社名、日付 |
| 2 | **アジェンダ（Agenda）** | 目次＋現在位置表示 | セクション一覧、ハイライト表示 |
| 3 | **セクション区切り（Section）** | 章の切り替え | セクション番号、セクション名 |
| 4 | **標準スライド（Default）** | 1メッセージ＋根拠 | キーメッセージ、ボディ、脚注、出所 |
| 5 | **2分割（TwoColumn）** | 左右比較、Before/After | 左カラム、右カラム、矢印（任意） |
| 6 | **3分割（ThreeColumn）** | 並列3要素の提示 | 3つのカード/カラム |
| 7 | **4分割（FourQuadrant）** | 2×2マトリクス | 4象限、X軸ラベル、Y軸ラベル |
| 8 | **プロセス（Process）** | 手順、フロー | ステップ矢印、各ステップの説明 |
| 9 | **タイムライン（Timeline）** | スケジュール、ロードマップ | 期間バー、マイルストーン |
| 10 | **比較表（ComparisonTable）** | 仕様比較、評価一覧 | テーブル、ハイライト行/列 |
| 11 | **KPI/数値ハイライト（KPI）** | 定量効果の強調 | 大きな数値、ラベル、変化率 |
| 12 | **チーム体制（Team）** | メンバー紹介 | 写真/アイコン、名前、役割、資格 |
| 13 | **エグゼクティブサマリー（Summary）** | 要約 | キーメッセージ一覧、構造化テキスト |

### デザイントークン（CSS変数として定義すべき値）

```css
/* カラーパレット */
--color-primary: #1B3A5C;     /* ネイビー（メインカラー） */
--color-secondary: #666666;   /* グレー（サブカラー） */
--color-accent: #E63946;      /* 赤（アクセント） */
--color-success: #2D9C4F;     /* グリーン（正の効果） */
--color-bg: #FFFFFF;           /* 白（背景） */
--color-text: #333333;         /* テキスト */
--color-muted: #999999;        /* 薄いグレー（補足テキスト） */

/* フォント */
--font-heading: 'Noto Sans JP', 'メイリオ', sans-serif;
--font-body: 'Noto Sans JP', 'メイリオ', sans-serif;
--font-number: 'Arial', 'Segoe UI', sans-serif;  /* 数値用欧文フォント */

/* フォントサイズ（16:9スライド基準） */
--font-size-title: 28px;      /* タイトル */
--font-size-subtitle: 20px;   /* 小見出し */
--font-size-body: 16px;       /* 本文（最低ライン） */
--font-size-caption: 12px;    /* 注釈（出所・脚注のみ） */
--font-size-kpi: 48px;        /* KPI数値 */

/* スペーシング */
--slide-padding: 40px;        /* スライド外周余白 */
--section-gap: 24px;          /* セクション間 */
--element-gap: 16px;          /* 要素間 */
--line-height: 1.5;           /* 行間 */
```
